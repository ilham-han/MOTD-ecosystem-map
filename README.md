# MOTD Ecosystem Map

A browser app for building speculative evolution and alien horror ecosystem maps.

## Open the app

Double-click:

```text
Open MOTD Ecosystem Map.cmd
```

Or open `index.html` in your browser.

## Publish online

See `DEPLOY.md` for simple Netlify, GitHub Pages, and Vercel publishing steps.

## What you can do

- Add organisms, creatures, plants, parasites, decomposers, and apex threats.
- Connect them with relationships like eats, infects, hosts, protects, and competes with.
- Drag nodes around the map to arrange your ecosystem.
- Edit selected organisms in the right inspector.
- Load the starter World of Ant ecosystem.
- Export your ecosystem as JSON and import it again later.

## Good next features
